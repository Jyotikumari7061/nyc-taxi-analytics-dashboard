#  Benefit Optimization Dashboard

## Tech Stack

**Frontend**: React 19, Tailwind CSS, Radix UI, CRACO  
**Backend**: FastAPI, MongoDB (motor), Pydantic v2  
**Dev Tools**: JWT Auth, AWS S3 (boto3), Pytest, Black, ESLint

## Installation

### Frontend
```bash
